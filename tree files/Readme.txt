Each file contains a list with 100 elements related to the 100 Rabosky's tree at 
genera ("listpcoafs.RData") and at species level ("listpcoafs.RData"). To read those
lists, you have to use the readRDS() function in R, for example, 
"readRDS("listpcoafs.RData")". Each element contains phylogenetic eigenvectors 
selected by the forward selection procedure, and we used them as phylogenetic 
explanatory variables in the analysis at genera and species level (see script available 
on Online Resource 1.1).
